# Programmer Calculator

## Descripción
**Programmer Calculator** es una calculadora diseñada específicamente para programadores. Permite realizar operaciones matemáticas y de conversión entre sistemas numéricos como binario, decimal, hexadecimal y octal.

## Instalación
1. Clona el repositorio en tu máquina local:
   ```
   git clone https://github.com/AlbertoTR00/programmer-calculator.git
   ```
2. Abre el archivo `index.html` en tu navegador para acceder a la calculadora de manera local.

## Uso
También puedes acceder a la aplicación directamente desde [GitHub Pages](https://albertotr00.github.io/programmer-calculator).

Simplemente abre el enlace y comienza a utilizar las funcionalidades que ofrece la calculadora.
